PinWindow

Force any window to be (or not to be) on top of other windows

Download this project as a .zip file Download this project as a tar.gz file

A small utility to make a window to be (or not to be) on top of other windows. No installation required.

Usage: Hold Win button and click anywhere in the window you want to pin on top of other windows. Use the same action (Win+left button click) to unpin a pinned window.
